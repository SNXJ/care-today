# UniApp 跨端版需求与验收标准

## 目标

在不改动现有 Vue Web 主线的前提下，新建 `uniapp/` 客户端，复用 Spring Boot REST API，支持构建 H5、微信小程序和 App 资源。

## 第一版范围

- 账号：手机号/邮箱密码登录、注册、登录态持久化、失效退出。
- 空间：创建陪伴空间、自动进入最近空间、查看成员、生成邀请链接。
- 五个主视图：今天、时间线、分享、身体、注意。
- 统一发布：分享、日程、身体评分、体温、体重、症状、注意事项、问医生、文本资料。
- 安全边界：隐私提示、医疗免责声明、敏感数据只经鉴权 API 访问。
- 更新：微信小程序使用 UpdateManager；App 使用远程版本清单，支持 APK 跳转和 WGT 热更新。

## 暂不包含

- 微信授权登录、订阅消息、系统级提醒。
- 图片/文件上传、评论点赞、AI 医疗问答。
- 平台管理后台、iOS App Store 自动发布。

## 验收标准

1. `npm run type-check` 无错误。
2. `npm run build:h5`、`npm run build:mp-weixin`、`npm run build:app` 均成功。
3. 未登录可登录或注册；无空间可创建；有空间可加载七类核心数据。
4. 五个 tab 可访问，统一发布页可提交九类记录。
5. 小程序启动时调用平台更新管理器；App 启动时读取远程版本清单，不在 H5 误触发。
6. API 地址、微信 AppID、DCloud AppID、更新清单地址均可配置，不在源码写入密钥。
